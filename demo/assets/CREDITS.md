# Photo credits

Every photograph in this folder is public domain or CC0. **No attribution is legally required** for
any of them — this file exists because provenance evaporates if nobody writes it down, and a future
reader deserves to know where these came from without re-doing the research.

Each licence below was read from the file's own Wikimedia Commons page (via the Commons API), not
taken on trust. Originals run up to 5760px; each was downscaled to fit 1400px and saved as JPEG so
the repository stays quick to clone. No other edit was made.
